Route Description for Users & Products

# Method Path Description
01 GET / Renders the home page (index.html)
02 POST /users/login Allows users to log into their account .
03 GET /users Returns an array displaying all users registered .
04 GET /users/:userID Returns an object of the user with matching ID .
05 PUT /users/:userID Updates details of the user with the matching ID .
06 POST /register Encrypts password given by user, saves user successfully in relevant database, and returns a JSON web token to the frontend.
07 DELETE /users/:userID Removes user from the relevant database .
08 GET /products Returns an array of all products .
09 GET /products/:id Returns an object of the selected product with matching ID .
10 POST /products Successfully saves product in relevant database .
11 PUT /products/:id Updates product in relevant database successfully -- only users with admin roles can update products .
12 DELETE /products/:id Deletes product from relevant database successfully -- only users with admin roles can update products .
